Pandas Basics 

This repository contains my beginner-level work on Pandas, one of the most powerful Python libraries used for data analysis and data manipulation.

About the Project:-
In this project, I explored the core concepts of Pandas using a Jupyter Notebook. The goal was to understand how to work with structured data efficiently and perform common data operations in a simple and practical way.

What I Learned:-
* Creating and working with **Series** and **DataFrames**
* Importing datasets (CSV, Excel, etc.)
* Data inspection and basic exploration
* Filtering and selecting data
* Handling missing values
* Basic data cleaning techniques
* Performing simple operations and transformations

Project Structure:- `pandas_basics.ipynb` → Main notebook with all the concepts and examples

 How to Run
1. Download or clone this repository
2. Open the notebook using:
   * Jupyter Notebook
   * JupyterLab
   * VS Code (with Jupyter extension)
3. Run the cells step by step to understand the workflow
